En este primer commit se realizaron dos documentos HTML, el primero llamado "index.html" que se encuentra enlazado a su hoja de estilo llamada "style-index-admin.css", la cual es la vista que se encontraran los administradores al iniciar sesión y por consiguiente no se debe editar hasta que cada uno de los modulos este realizado, el segundo y mas relevante en este momento es llamado "plantilla.html" con su respectivo documento CSS llamado "style.css" el cual se realizo con el fin de que pueda ser reutilizado y enlazado en los respectivos documentos HTML de cada modulo usando la plantilla anteriormente nombrada, cabe recalcar que en el proyecto hasta este punto se utilizo un solo script llamado "script.js" en el que se encuentra el algoritmo implementado en ambos documentos con el fin de darle funcionalidad a la barra lateral que se logra apreciar desde la vista para dispositivos moviles.

A continuacion encontraran unas lineas de codigo que pueden ser de gran utilidad conforme se vaya avanzando en el proyecto y a su vez facilitaran un correcto enlace entre las distintas partes del proyecto.

	* color naranja originario del SENA: #ff8138
	* fuente utilizada: font-family: 'Open Sans', sans-serif;
	* color utilizado para los bordes en fondo claro: 1px solid rgba(51, 51, 51, 0.103)
	* color utilizado para los bordes en fondo oscuro: 1px solid rgba(255, 255, 255, 0.5)
	* color utilizado para la fuente de letra: color: rgb(58, 58, 58);
	* negrita recomendada: font-weight: 400;

Cabe ratificar que esta es la primera version del proyecto y por lo tanto es normal que se vean algunos bug durante la codificación, asi mismo como los iconos hasta el momento estan siendo llamados desde la red y no estan descargados dentro del proyecto asi como tambien la fuente, de igual manera puede que se encuentre codigo repetido, de todos modos se recomienda no alterar el codigo ya existente ya que esto se hara en un futuro y a su vez para no causar fallos.
